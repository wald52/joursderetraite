# Calculatrice d'Équivalent Retraites - PWA

Cette application est une Progressive Web App (PWA) qui permet de convertir des montants en équivalent temps de retraites versées en France.

## Installation des icônes

Pour que l'application PWA fonctionne correctement, vous devez créer les fichiers d'icônes suivants :

- `icon-192x192.png` : Icône de 192x192 pixels
- `icon-512x512.png` : Icône de 512x512 pixels

Vous pouvez créer ces icônes à partir du fichier `icon.svg` fourni en utilisant un outil de conversion SVG vers PNG.

## Fonctionnalités PWA

- Installation sur l'écran d'accueil
- Fonctionnement hors ligne (pour les pages de base)
- Thème couleur correspondant à l'application
- Affichage en mode standalone

## Utilisation

1. Ouvrez l'application dans un navigateur moderne (Chrome, Firefox, etc.)
2. Une fois que le service worker est enregistré, vous verrez une notification d'installation
3. Suivez les instructions pour installer l'application sur votre appareil
